# Welcome to MovieSite

MovieSite was made over a two-day period during Fullstack Academy of Code's Stackathon. It is a MVP movie search tool made with React, Redux, Bootstrap, and the Movie Databaseâ€™s API.

## Installation
First, clone the repository through git and change to the new directory:
```
git clone https://github.com/
cd moviesite
```
Then install the required dependencies:
```
npm install
```
Run the app:
```
npm start
```
